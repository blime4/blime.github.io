[hexo已解决] Error: ENOSPC: System limit for number of file watchers reached
2020-06-20|Tutorial
输入hexo cl && hexo g && hexo s的时候 突然报出这个错误 Error: ENOSPC: System limit for number of file watchers reached吓得我按照他的指示，删了很多空文件夹都解决不了 然后百度得知这是因为 是文件监视程序的系统产生了限制，达到了默认的上限，需要增加限额永久增加限额 然后找到了解决方案 在终端按顺序执行下面两个命令即可解决问题 echo fs.inotify.max_user_watches=524288 | sudo tee -a /etc/sysctl.conf && sudo sysctl -psudo sysctl --system
